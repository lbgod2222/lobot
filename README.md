# lobot

### 驱动树莓派达到功用目的

- 选用Jonnyfive作为通讯IO
- 本仓库仅作为测试仓库，前端仅是测试项目
- 阿里云服务节点作为内网穿透服务器
- 树莓派4B作为操作对象机


### 功用列表

- 摄像机云台舵机控制
- 流媒体传输
- 仿生操作控制
